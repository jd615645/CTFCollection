Installation:
Setup your web server the serve this directory.


Requirements:
None


Usage:
Problems are organized by year and contest. When adding or removing a problem, simply get a copy of the example folder from the root folder and put it where it belongs. The title of a problem or a contest is a combination of alphabets and number and dashes. It must not contain any underlines and spaces.

Run “python3 update.py” to update problem index.
